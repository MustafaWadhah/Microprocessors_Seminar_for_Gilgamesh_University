# Interactive Microprocessors Basics Seminar

![Microprocessor Representation](/public/images/amd_cpu.png)

A highly stylized, fully interactive single-page web presentation covering the foundational concepts of Microprocessors. This open-source engine replaces traditional, static slideshow presentations (like PowerPoint) with a breathtaking **liquid-glass, neon-cyberpunk aesthetic** built purely in Vanilla Javascript, HTML5, and CSS3.

## ✨ Features
* **Modern Glassmorphism UI:** Seamless 16:9 responsive display with gorgeous blur drops and dynamic shadow casting.
* **3D Slide Transitions:** Smooth rotating depth-transitions built entirely in CSS `transform`.
* **Live Hardware Emulators:** 
  * Playable Logic Gates (AND, OR, NAND, XOR) with physical LED outputs.
  * Step-by-step Assembly Simulator.
  * Step-by-step PDP-11 Architecture loop simulator.
  * Real-time architectural Instruction Cycle animator.
* **Motherboard Trace Background:** Procedural algorithms generating data-traces pulsing out across your entire screen randomly in real-time.

## 🚀 How to Run Locally

This project uses [Vite](https://vitejs.dev/) for blazing-fast local development and production bundling.

1. Clone this repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open the provided `localhost` URL in your browser!

To build for production, run `npm run build`.
